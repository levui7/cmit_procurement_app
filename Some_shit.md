**ЛОГИКА РАБОТЫ ПРОГРАММЫ**

0. Запуск приложения и офигенное анимированное лого 

!!! ПОЛЬЗОВАТЕЛЬ !!!
1. Ввод данных пользователем
Пользователь открывает приложение и вводит начальные данные:
- Примерное количество учеников на год
- Период обучения

Также пользователь вводит параметры для фильтрации:
- Рейтинг товара на маркетплейсе
- Дата доставки
- Цена
!!! ПОЛЬЗОВАТЕЛЬ !!!

!!! РАБОТА ПРОГРАММЫ !!!
2. Рассчет потребностей
Рассчет необходимого количества для закупок с учетом поломок (нпа основе Внутреннего каталога).
Получаем список позиций с количеством для закупки.

3. Поиск и маппинг данных
Нужно найти позиции на маркетплейсах.
- Поиск товара в БД -> использование ключевых слов для поиска
- Парсер собирает товары по запросу
- Фильтрация: отсеить товары, не подходящие по смыслу; фильтрация по рейтингу; сортировка по цене
- Отбор подходящих вариантов по каждой позиции
!!! РАБОТА ПРОГРАММЫ !!!

!!! ПОЛЬЗОВАТЕЛЬ !!!
4. Интерфейс выбора товаров
Таблица редактирования наименований
В виде таблицы с отзывами, рейтингом, ценой и ссылкой
Пользователь нажимает на кнопку ВЫБРАТЬ напротив избранного варианта
