## **Десктопное приложение для автоматизации закупок материалов для предприятия ЦМИТ ЛЮКС**.
Приложение показывает всех актуальных поставщиков. По заданным условиям приложение показывает идеальный вариант для закупки. Есть возможность посмотреть отзывы пользователей о товаре.

Плановая закупка на 1 год — 1 раз в год; возможны быстрые доп. закупки в случае поломок.
Около 60 чел. обучаются в год.

15-20% материалов ломаются, учитывать поломки!

Прогноз: сколько человек будет заниматься, сколько товаров и где закупать.  

Цель: разработать единый интерфейс закупки - десктоп приложение
В интерфейсе должны быть:
- список того что закупить по рубрикам (прописать наименования и возможность редактирования наименований, как состыковать наименования в приложении с тем, что есть на рынке).
- интерфейс прогнозного количества учеников 
- исходя из этого количества с учетом количества поломок должен быть сформирован список на закупку
